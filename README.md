# PROT-TIPO-PARA-CAPTURA-E-TRANSMISS-O-DE-IMAGENS-DE-BAIXA-RESOLU-O-UTILIZANDO-CAT-NB2
Protótipo embarcado para captura e transmissão de imagens de baixa resolução via rede Cat NB2, com foco em baixo consumo energético e conectividade IoT. O sistema utiliza câmera compacta, compressão de dados e envio eficiente para monitoramento remoto em áreas com cobertura limitada.
